네이버 API 자격증명 발급 안내

네이버 검색광고 API 및 쇼핑 오픈 API는 아래 순서에 따라 발급받을 수 있습니다.

1. 검색광고 API 발급 (캠페인/광고그룹/입찰 연동)

  1. ① 네이버 검색광고 센터 로그인
  2. ② 상단 메뉴 [도구] - [서비스 API/광고 API] 클릭
  3. ③ 라이선스 동의 및 키 생성 진행
  4. ④ 발급된 고객 ID, 액세스 API Key, Secret Key, 라이선스 키를 기입

2. 쇼핑 검색 오픈 API 발급 (실시간 순위/경쟁사 매칭)

  1. ① 네이버 개발자 센터 로그인 후 [Application 등록]
  2. ② 사용 API에서 [검색]을 추가
  3. ③ 비로그인 오픈 API 서비스 환경에 [WEB 설정] 추가
  4. ④ 웹 주소는 임의의 URL(예: http://localhost:3000) 등록
  5. ⑤ 발급된 Client ID 및 Client Secret을 입력
🔒

보안 안내

입력한 모든 API Key는 백엔드 서버(`.env` 및 로컬 세션)에만 기록되며, 외부로 유출되거나 브라우저 소스코드 상에 노출되지 않도록 서명은 서버(Express/Cloudflare Worker)에서 안전하게 연산됩니다.
